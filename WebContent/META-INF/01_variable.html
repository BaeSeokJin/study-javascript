변수 선언

	01. 자바스크립트는 인터프리터 언어
	02. 인터프리터(통역사) : 개발자가 작성한 소스코드를 컴파일하는 과정 없이 바로 실행
	03. 자바스크립트는 '타입추론'을 지원하는 언어
	
	* 자바스크립트의 변수 선언 *
	1) var 변수명; ES6 이전
	2) let 변수명; ES6 이후 ***
	3) const 상수명; ES6 이후 ***
	
	  var : 재선언(o), 재할당(o), 함수레벨 스코프, 변수 호이스팅 가능
	  let : 재선언(x), 재할당(o), 블록레벨 스코프, 변수 호이스팅 불가능
	const : 재선언(x), 재할당(x), 블록레벨 스코프, 변수 호이스팅 불가능
	
	* 타입 추론 : 변수에 할당된 값에 따라 변수의 타입이 변경
	* 변수 호이스팅 : 변수를 선언하지 않고 값을 할당할 수 있다
	* 재선언 : 같은 이름의 변수를 다시 선언
	* 함수레벨 스코프 : 변수가 자신이 선언된 함수가 종료될 때 메모리에서 삭제되는 것
	
	* 변수 호이스팅(var만 가능)
		자바스크립트 앤진이 코드를 해석할 때 변수나 함수를 선언하는 코드를 먼저 해석하고,
		이후 초기화 코드를 해석한다. 따라서 코드 순서상으로 변수나 함수를 선언하는 코드가 뒤에 있더라도
		해석 순서상 먼저 선언부를 해석하기 때문에 코드 실행에는 지장이 없다.
