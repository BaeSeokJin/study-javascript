연산자

	산술 : +,-,*,/,%
	단항 : ++,--
	비교 : <,>,<=,>=,==,===, !=,!==
	논리 : &&, ||, !
	삼항 : 조건식?값1:값2

==, ===

	==(동등연산자) : 값만 비교
	===(일치연산자) : 값과 타입을 모두 비교

자바스크립트에서의 값 -> boolean 값

	0                       => false
	                        => false
	undefined               => false  
	null                    => false
	NaN(not a number)       => false
	나머지 값은 모두 true로 반환
